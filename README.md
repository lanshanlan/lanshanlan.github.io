# Alex Chen - 个人简历

## 项目介绍

这是一个现代化的个人简历网站，展示了Alex Chen的个人信息、技术栈、工作经历、爱好和相册。网站采用了响应式设计，具有3D背景效果和平滑的滚动动画，为访客提供了流畅的浏览体验。

## 技术栈

- **前端基础**：HTML5, CSS3, JavaScript (ES6+)
- **框架**：React, Vue, Next.js
- **样式**：Tailwind CSS, SASS, Styled Components
- **工具**：Webpack, Git, Figma, Adobe XD
- **3D/动效**：Three.js, GSAP, Framer Motion
- **后端**：Node.js, Express, MongoDB

## 项目结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # JavaScript文件
└── README.md           # 项目说明文档
```

## 功能特点

1. **响应式设计**：适配不同屏幕尺寸，从手机到桌面设备
2. **3D背景效果**：使用Three.js实现动态粒子和线条效果
3. **平滑滚动**：点击导航链接时的平滑滚动效果
4. **滚动动画**：页面滚动时的元素入场动画
5. **交互式卡片**：技术栈、爱好等卡片的悬停效果
6. **时间线布局**：工作经历的时间线展示
7. **相册网格**：个人照片的网格布局

## 运行项目

1. 克隆项目到本地
2. 进入项目目录
3. 启动本地服务器（推荐使用Python的内置HTTP服务器）：
   ```bash
   python3 -m http.server 8000
   ```
4. 在浏览器中访问：http://localhost:8000
